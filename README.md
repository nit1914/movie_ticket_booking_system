# 🎬 Movie Ticket Booking System

A simple desktop-based Movie Ticket Booking System built using **Python Tkinter** and **MySQL**.

---

## 📌 Overview

This application allows users to:
- Select a movie show
- Enter their name
- Reserve seats automatically
- View available seats
- Track booking history

The system updates seat availability in real time and stores booking data in a MySQL database.

---

## 🚀 Features

- 🎬 Multiple movie shows
- 🪑 Automatic seat allocation (row & seat)
- 📊 Live seat availability tracking
- 📜 Booking history display
- 💾 MySQL database integration
- 🖥️ GUI using Tkinter

---

## 🛠️ Tech Stack

- **Frontend:** Tkinter (Python GUI)
- **Backend:** Python
- **Database:** MySQL
